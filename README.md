# Space-Shooter Game
# Game Description
Keybord control for PLAYER 1 :

    W - UP
    S - DOWN
    A - LEFT
    D - RIGHT
    C - to shoot, Use 'w' and 's' to change direction.

Keybord control for PLAYER 2 :

    I - UP
    K - DOWN
    J - LEFT
    L - RIGHT
    M - to shoot, Use 'I' and 'K' to change direction.
